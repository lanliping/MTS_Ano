# 使用说明

## 运行环境
```
python 3.8+
pip install pysimdjson
```

## 运行命令
```
python main.py
```

## 修改说明
1. 数据放置到正确位置 ```.../Dataset/"dataset"/download-version/"data_file"```
2. 修改main文件中参数，三个参数都需要修改。数据存储地址统一使用```.../Dataset/"dataset"/json-version/"data_file"```
3. 自定义```"dataset".py```，修改两个函数中的具体功能（不要修改函数名称）
